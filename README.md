# Cinema Room REST Service

A Spring Boot application that provides RESTful services for managing a cinema room.

## Description

This project is a REST API for a cinema room management system. It allows users to interact with cinema room functionalities such as booking seats, viewing available seats, and more.

## Prerequisites

- Java 25
- Gradle

## Getting Started

1. Clone the repository:
   ```
   git clone <repository-url>
   cd cinema-room-rest-service
   ```

2. Build the project:
   ```
   ./gradlew build
   ```

3. Run the application:
   ```
   ./gradlew bootRun
   ```

The application will start on `http://localhost:8080`.
